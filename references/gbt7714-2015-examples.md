# GB/T 7714-2015 Citation and Reference Examples

All generated reports, citations, and reference lists MUST strictly follow GB/T 7714-2015. If bibliographic metadata is incomplete, mark missing fields explicitly instead of fabricating them.

**These examples are formatting aids, not a substitute for the official GB/T 7714-2015 standard.**

Reference: GB/T 7714-2015 Information and documentation — Rules for bibliographic references and citations.

---

## General Format

```
责任者. 题名[文献类型标识]. 出版地: 出版者, 出版年.
```

---

## 1. Journal Article [J]

English journal:
```
HE K, ZHANG X, REN S, et al. Deep residual learning for image recognition[J]. IEEE Transactions on Pattern Analysis and Machine Intelligence, 2016, 39(6): 1173-1180.
```

With DOI:
```
LECUN Y, BENGIO Y, HINTON G. Deep learning[J]. Nature, 2015, 521(7553): 436-444. DOI: 10.1038/nature14539.
```

Chinese journal:
```
张三, 李四. 基于深度学习的小目标检测方法研究[J]. 计算机工程与应用, 2022, 58(10): 1-10.
```

Example only; replace with verified bibliographic metadata.

## 8. Dataset / Benchmark

As a published paper:
```

```

As an online dataset resource:
```

```

```
VISDRONE. VisDrone dataset[EB/OL]. [2024-01-15]. https://github.com/VisDrone/VisDrone-Dataset.
```

## 9. Multi-author English Format

- List up to 3 authors by name, then add `et al.`
- Format: `SURNAME Initial, SURNAME Initial, SURNAME Initial, et al.`
- Example: `HE K, ZHANG X, REN S, et al.`

## 10. DOI Handling

- Include DOI when available: `DOI: 10.xxxx/xxxxx`
- Place after the page numbers or volume/issue info

## 11. URL and Access Date

- For online resources, include access date in square brackets: `[2024-06-01]`
- URL follows the access date

## 12. In-text Citation Format

Use numbered citations in square brackets: `[1]`, `[2-5]`, `[1,3,7]`

When referencing specific sections: `as shown in Section 3.2 [1]`

## 13. Missing Field Handling

When a field is unavailable from the provided material, use explicit markers:

- Missing author: `Missing metadata: author`
- Missing year: `Missing metadata: year`
- Missing publisher: `Missing metadata: publisher`
- Missing page numbers: `Missing metadata: pages`
- Missing DOI: omit
- Missing volume/issue: omit
- Missing URL: omit

**Never fabricate any bibliographic field.** If metadata is incomplete, mark missing fields explicitly rather than guessing or inventing values.
